Form Inputs.

Forms contain multiple things like Text Input, Password, Checkbox etc, we'll see them below along with their validation.

Inputs with/without required attribute

By default anyone can submit a form if there is no required attribute on it.

So, we add a required attribute for those inputs which are needed. Also add form-input-required class to the label to highlight that input is needed.

Disabled and Read only.

To disable an input, just add disabled attribute to the tag, same holds for read only, just add readonly.
