Image.

Images can be made responsive, so that they can adjust to the parent's width and never overflow. They can also be customsed to round shape.

Responsive Image.

To make an Image responsive, just add image-responsive class to image tag.

Round Image.

To make an Image round, just add image-round class to image tag.

This image-round has a default Image wifth of 250px.
