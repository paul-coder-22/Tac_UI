Buttons

A Button is a simple element that helps in performing an action/redirection.

Primary button styles.

Primary Styles are the ones that match the whole theme of your website.

Link Button

Secondary button styles.

You can add icons to secondary buttons as well.

Secondary outline button styles.

Button sizes.

We have 3 different sizes. Small, Normal-which is the default & Large. Just apply btn-size-sm for a smaller button and btn-size-lg for a larger one.

Icon button styles.

Icon button sizes.

Icons are of three sizes, Small-sm, normal-default, Large-lg, just apply btn-icon-sm for a smaller icon and btn-icon-lg for a larger one.

Floating Buttons.

These are used to make user experience faster & easier.

To get this, just add floating-btn class to your <button> .
