Avatar

An avatar is a graphical representation of a user or the user's character or persona.

Just add avatar class to a parent element of image tag and you are good to go.

Round Avatar.

This can be achieved by adding avatar-round class to the parent element of avatar image

Square Avatar.

This can be achieved by adding avatar-square class to the parent element of avatar image

Avatar Sizes.

This can be achieved by adding avatar-xl, avatar-large, avatar-medium or avatar-small class to the parent element of avatar image

Text Avatar.

This can be achieved by adding avatar-text class to the parent element of avatar image
