Text Utilities

Text Utilities are used to style an element that hold some text, easily.

Headings.

These are default headings, can be achieved by wrapping the content in <h1>, <h2>, <h3>, <h4>,<h5> and <h6>.

Text Alignment.

We have three different text alignments, Left, Center and Right.

To achieve this, just add txt-left, txt-center, txt-right class to the element.

Text Decorations.

We have few text styles, to achieve any of them just added their respective class to the element.

The classes are txt-strike, txt-underline, txt-uppercase, txt-grey, txt-bold .

Coloured Texts.

There are five text colors, to achieve each of them just added their respective class to the element.

The classes are txt-clr-primary, txt-clr-error, txt-clr-warning, txt-clr-info, txt-clr-success.

Text Sizes.

We have five font sizes, to achieve each of them just added their respective class to the element.

The classes are font-size-xs, font-size-sm,font-size-normal, font-size-lg, font-size-xl.
